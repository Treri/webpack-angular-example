### webpack-example

Use Angular, Promise, Webpack and ES2015 to write async load angular code.
And only load specific code on demand.

#### Used libraries or tools

- bower
- angular.js
- angular-ui-router.js
- promiz.js
- webpack
- then-loader
- style-loader
- css-loader

#### Run

```bash
npm install
bower install
npm run webpack
npm run serve
```
Now open <http://localhost:3000/>
